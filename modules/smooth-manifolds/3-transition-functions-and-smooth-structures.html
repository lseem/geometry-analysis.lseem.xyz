3. Transition functions and smooth structures

In this section we will define what it means for a topological manifold to have a smooth structure. This smooth structure allows us to differentiate functions on the manifold; indeed, it defines what is means for a function on the manifold to be smooth.

Warning

The following is a bit of a dead end.

It is worth taking five minutes here to see if you can solve this problem yourself. That is, for a topological manifold $X$, think about how you could define what it means for a function $f:X\to\R$ to be smooth.

Consider a topological manifold $X$, and let $f:X\to\R$ be continuous. The key here is to consider charts $(U,\phi)$ and $(V,\psi)$ such that $U\cap V\neq\empty$ (i.e. the charts overlap).

Then the compositions $f\circ\inv{\phi}$ and $f\circ\inv{\psi}$ are functions from a subset of $\R^n$ to $\R$. They might be differentiable! Let’s say both the compositions are differentiable. Then which chart should we use to define whether or not $f$ is differentiable? Does it matter which one we choose? This question is:

Question: Let $f:X\to\R$ be continuous, and let $(U,\phi), (V\psi)$ be overlapping charts. Is it the case that $f\circ\inv{\phi}$ is differentiable if and only if $f\circ\inv{\psi}$? Answer: No. Take, for example, $X=\R$, $f=\id$, $\phi=\id$, $\psi(x)=x^3$. Then the compositions $f\circ\phi^{-1}=\id$ and $f\circ\psi^{-1}$ which is differentiable.
